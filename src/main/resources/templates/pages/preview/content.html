1. 목표 정의

페르소나, 문제, 톤을 정의한 후 초안 생성 버튼을 누르세요. 토이GPT가 이를 바탕으로 글뼈대를 만들고 중요한 메시지를 강조합니다.

2. 초안 생성

AI가 작성한 첫 번째 초안을 받아 미리보기에서 확인합니다. 코드, 이미지, 인용구 등을 자동으로 서식화해 줍니다.

3. 스타일 맞춤

스타일 패널에서 버튼 스타일, 표지 카드, CTA를 수정하고 실시간으로 반영합니다.

4. SEO 체크

메타 설명, 키워드, 링크 구조를 점검하는 체크리스트가 자동 표시됩니다.

5. 배포와 공유

발행 버튼을 누르면 스타트부트스트랩형 포스트 레이아웃으로 배포되고 SNS 공유 링크가 생성됩니다.
